# 씬 담당(Scene Owners)

> Unity 씬(.unity)·프리팹은 여러 명이 동시에 건드리면 충돌이 심하다.
> **씬 파일은 아래 담당자만 편집**하고, 다른 사람은 자기 작업을 **prefab으로 만들어 담당자에게 넘긴다.**

## 씬 담당표

| 씬 | 담당자 | 비고 |
|----|--------|------|
| `BootScene` |  | 공용 매니저 초기화 |
| `TitleScene` |  | 새 게임/이어하기/옵션 |
| `LoadingScene` |  | 비동기 로드·진행률 |
| `InGameScene` |  (Scene Owner) | 1차 빌드는 이 씬 하나로 (거점/월드맵/무역도시 전환) |

> 1차 빌드는 기획서 위험요소 #8에 따라 **InGameScene 1개**로 화면을 전환한다.
> 콘텐츠가 커지면 그때 씬을 분리한다.

## 씬 편집 규칙

1. 씬을 바꾸기 전에 **Scene Owner에게 알린다**(같은 씬 동시 편집 금지).
2. 내 기능은 **prefab으로 만들어** 씬에는 배치만 요청한다.
3. 씬 변경 PR은 **작게, 자주** 올린다(오래 열린 브랜치 금지).
4. 씬 충돌이 나면 → UnityYAMLMerge 시도 → 확신 없으면 **Scene Owner 버전 기준**으로 선택, 잃은 변경은 Unity에서 다시 적용. **`.meta` 삭제로 해결하지 않는다.**
