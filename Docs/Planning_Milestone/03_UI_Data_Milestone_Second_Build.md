## 4. 거점 도시 UX 확장 가능성

향후 거점 도시 UX 확장 기획이 추가될 경우 Content & Tools와 공동으로 다음 작업을 담당할 수 있다.

- 건물 상세 정보와 상호작용
- 도시 성장 단계 시각화
- 건물 배치·선택 UX
- 거점 상태와 무역 준비 연결
- 해금·생산·도시 효과 표시

현재 2차 확정 범위에는 최소 건물 비용·효과 UI만 포함하며, 추가 확장은 별도 기획 확정 후 마일스톤에 편입한다.

## 5. 완료 기준

- 모든 최종 기능을 플레이어가 제품 UI에서 조작한다.
- UI 내부에 경제·Core 계산을 중복 구현하지 않는다.
- 최종 기간에 대규모 화면 구조 변경이 필요하지 않다.

## 6. 일정 지연 시 축소

- 애니메이션 제외
- 기부·투자·대출은 각 단일 화면
- 추천 정보 1종
- 거점 추가 UX는 후속 기획으로 유지
