# Core Gameplay 담당자 — 2차 빌드 개인 마일스톤
## 선택 A — 최종 기획 범위 유지형

## 1. 역할

Caravan 구성, 출발 검증, 이동·식량·내구도·용병·이벤트 결과, 성공·실패와 기부·투자·대출 효과의 런타임 반영을 담당한다.

- **2차 빌드 개발 기간:** 2026-07-20 ~ 2026-08-06
- **2차 빌드 작업일:** 14일
- **최종 빌드 개발 기간:** 2026-08-07 ~ 2026-08-11
- **최종 빌드 작업일:** 3일
- **주말 및 공휴일:** 작업을 배정하지 않는다.
- **1차 빌드 기획:** 2026-07-17 확정본을 고정하며 소급 수정하지 않는다.
- **기준 브랜치:** `dev2`

- **구조 동결:** 2026-08-04 업무 종료 시점
- **기능 동결:** 2026-08-05 업무 종료 시점
- **2차 빌드 RC:** 2026-08-06
- **최종 빌드 기간:** 회귀 테스트, Blocker 수정, 밸런스, 콘텐츠 누락 보완, 연출, 제출 안정화만 수행한다.

## 4. 완료 기준

- 제품 UI에서 받은 명령만으로 전체 루프가 동작한다.
- 기부·투자·대출 효과가 런타임에 실제 영향을 준다.
- 재실행 후 파생값과 해금 상태가 올바르게 복구된다.
- 최종 기간에 Core 상태 구조 변경이 필요하지 않다.
