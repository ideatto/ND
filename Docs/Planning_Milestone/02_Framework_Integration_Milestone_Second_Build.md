# Framework & Integration 담당자 — 2차 빌드 개인 마일스톤
## 선택 A — 최종 기획 범위 유지형

## 1. 역할

씬, 시간, 저장, 오프라인 진행, 이벤트, 기부·투자·대출 상태 복구, 통합 빌드와 RC를 담당한다.

## 공통 일정 원칙

- **2차 빌드 개발 기간:** 2026-07-20 ~ 2026-08-06
- **2차 빌드 작업일:** 14일
- **최종 빌드 개발 기간:** 2026-08-07 ~ 2026-08-11
- **최종 빌드 작업일:** 3일
- **주말 및 공휴일:** 작업을 배정하지 않는다.
- **1차 빌드 기획:** 2026-07-17 확정본을 고정하며 소급 수정하지 않는다.
- **기준 브랜치:** `dev2`
- **통합 방식:** 기능 브랜치에서 작업하고 PR을 통해 `dev2`에 반영한다.
- **2차 빌드 목표:** 최종 기획 기능의 약 90%를 구현하고, 최종 빌드 기간에는 신규 대형 시스템 개발을 금지한다.
- **구조 동결:** 2026-08-04 업무 종료 시점
- **기능 동결:** 2026-08-05 업무 종료 시점
- **2차 빌드 RC:** 2026-08-06
- **최종 빌드 기간:** 회귀 테스트, Blocker 수정, 밸런스, 콘텐츠 누락 보완, 연출, 제출 안정화만 수행한다.
