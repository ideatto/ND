# 누워서 돈벌기

> **1차 빌드 안내 문서**
> 장르: 방치형 무역 경영 시뮬레이션

## 1. 게임 소개

**누워서 돈벌기**는 중세 판타지풍 세계를 배경으로 하는 방치형 무역 경영 시뮬레이션 게임입니다.

플레이어는 작은 거점을 기반으로 하나의 **Caravan**을 운영합니다. 도시와 무역로를 선택하고, 마차에 무역품과 식량을 적재한 뒤 견인 동물을 배치하여 다른 도시로 무역을 떠나야 합니다.

출발한 Caravan은 시간의 흐름에 따라 목적지로 이동합니다. 플레이어는 도착 후 판매 수익과 비용을 정산하고, 다시 다음 무역을 준비합니다.

1차 빌드는 다음 핵심 흐름을 플레이어 UI로 반복할 수 있는지 검증하는 버전입니다.

```text
Caravan 준비
→ 무역 출발
→ 이동 진행
→ 성공 또는 실패 정산
→ 보상 수령
→ 다음 무역 준비
```

---

## 2. 주요 조작법

1차 빌드의 주요 기능은 **마우스 클릭 중심**으로 조작합니다.

### 타이틀 화면

| 메뉴                          | 기능                                               |
| --------------------------- | ------------------------------------------------ |
| **새 게임(New Game)**          | 새로운 저장 데이터로 게임을 시작합니다.                           |
| **이어하기(Continue)**          | 저장된 무역 상태와 진행 상황을 불러옵니다. 저장 데이터가 없으면 사용할 수 없습니다. |
| **저장 초기화(Reset Save Data)** | 확인 창을 거쳐 기존 저장 데이터를 삭제합니다.                       |
| **종료(Exit)**                | 게임을 종료합니다.                                       |

### Caravan 준비 화면

Main UI 에서 중앙의 '무역' 버튼을 누르면 무역을 시작합니다. 

1. 목적지 도시와 무역로 선택
2. 구매하거나 운송할 무역품 선택
3. 식량 적재
4. 마차 선택
5. 견인 동물의 종류와 수량 선택
6. 현재 적재량, 적정 적재량, 최대 적재량 확인
7. 출발 가능 여부와 차단 사유 확인
8. **무역 출발** 버튼 클릭

잘못된 구성이면 출발할 수 없으며, 화면에 출발 불가 사유가 표시됩니다.

### 이동 화면

무역 출발 후 다음 정보를 우측 상단의 지도 버튼을 눌러 확인할 수 있습니다.

* 현재 이동 진행률
* 이 무역루트의 위험도
* Caravan이 이동하는 장면

진행 중 게임을 종료하더라도 저장된 시간과 Caravan 상태를 기준으로 이어하기가 가능합니다.

### 정산 화면

무역이 완료되거나 실패하면 다음 결과를 확인합니다.

* 무역 성공 또는 실패
* 판매 수익
* 구매 비용 및 기타 적용 비용
* 최종 순이익
* 실패한 경우 실패 원인

결과를 확인한 뒤 **정산 받기(Claim)** 버튼을 누르면 재화가 반영되고 다음 무역 준비 화면으로 돌아갑니다.

---

## 3. 1차 빌드 플레이 루프

### 1단계: 새 게임 또는 이어하기

처음 실행했다면 **새 게임**을 선택합니다.

이전에 무역을 진행한 저장 데이터가 있다면 **이어하기**를 선택하여 준비, 이동 또는 정산 상태를 복구할 수 있습니다.

### 2단계: 목적지와 무역로 선택

거점에서 이동할 도시와 무역로를 선택합니다.

선택한 무역로는 Caravan의 이동 거리와 소요 시간에 영향을 줍니다.

### 3단계: 무역품과 식량 적재

판매할 무역품과 이동 중 소비할 식량을 마차에 적재합니다.

식량도 마차의 적재 공간을 사용하므로, 상품과 식량의 비율을 적절히 조절해야 합니다.

### 4단계: 마차와 견인 동물 구성

마차와 견인 동물을 선택합니다.

* 마차마다 적정 적재량과 최대 적재량이 다릅니다.
* 견인 동물은 이동속도, 식량 소모량, 적정 적재량에 영향을 줍니다.
* 한 Caravan에는 서로 다른 종류의 견인 동물을 섞을 수 없습니다.
* 마차가 요구하는 최소 견인 동물 수를 충족해야 합니다.

하지만 현재 1차 빌드에서는 두다리로 손수레를 끌어야합니다. 

### 5단계: 출발 조건 확인

화면에 표시된 적재량과 출발 가능 여부를 확인합니다.

| 적재 상태     | 조건                      | 결과               |
| --------- | ----------------------- | ---------------- |
| **적정 적재** | 현재 적재량이 최종 적정 적재량 이하    | 정상 속도로 출발        |
| **과적**    | 최종 적정 적재량 초과, 최대 적재량 이하 | 출발 가능하지만 이동속도 감소 |
| **적재 불가** | 최대 적재량 초과               | 출발 불가            |

### 6단계: 무역 출발 및 이동

출발에 성공하면 Caravan이 목적지로 이동합니다.

식량은 인게임 경과 시간에 따라 소비되며, 진행 중 게임을 종료해도 재실행 시 경과 시간이 반영됩니다.

### 7단계: 성공 또는 실패 정산

목적지에 정상 도착하면 판매 수익과 비용을 계산합니다.

무역에 실패한 경우에도 정산 화면으로 이동하며, 실패 원인과 결과를 확인할 수 있습니다.

### 8단계: 정산 수령 및 재출발

**정산 받기(Claim)**를 눌러 결과를 확정합니다.

정산이 끝나면 Caravan 준비 화면으로 돌아가 다음 무역을 시작할 수 있습니다.

---

## 4. 출발할 수 없는 조건

다음 조건은 **무역 실패가 아니라 출발 전 구성 오류**입니다. 조건을 수정하기 전까지 무역을 시작할 수 없습니다.

* 목적지, 무역로, 마차 등 필수 구성이 누락된 경우
* 마차의 아이템 슬롯 또는 아이템 중첩 한도를 초과한 경우
* 마차에 무역품이 하나라도 실려 있지 않은 경우 
* 현재 적재량이 마차의 최대 적재량을 초과한 경우
* 서로 다른 종류의 견인 동물을 한 Caravan에 함께 배치한 경우
* 마차가 요구하는 최소 견인 동물 수를 충족하지 못한 경우

최종 적정 적재량을 초과했더라도 최대 적재량 이하라면 **과적 상태로 출발할 수 있습니다.** 단, 이동속도가 감소합니다.

---

## 5. 무역 실패 조건

### 1)식량 부족

1차 빌드에서 플레이 중 발생하는 핵심 무역 실패 조건은 **식량 부족 상태의 지속**입니다.

```text
이동 중 식량 소모
→ 남은 식량이 0이 됨
→ 식량 부족 상태 진입
→ 부족 상태가 허용 시간 이상 지속
→ 무역 실패
→ 실패 정산
```

식량이 0이 되는 순간 즉시 실패하지는 않습니다.

식량 부족 상태가 일정 시간 이상 지속되면 무역 실패가 확정됩니다. 실패하더라도 게임이 종료되지는 않습니다. 정산 결과를 수령한 뒤 Caravan을 다시 구성하여 다음 무역을 시작할 수 있습니다.

**2)내구도 소모**

1차 빌드에서 이동에 따라 마차의 내구도가 감소합니다.

```text
이동 진행
→ 내구도 감소
→ 내구도가 0이 됨
→ 내구도 부족 상태 진입
→ 부족 상태가 허용 조건을 초과
→ 무역 실패
→ 실패 정산
```

내구도가 0이 되는 순간 즉시 실패하지는 않습니다.

내구도 부족 상태가 일정 조건을 초과하면 무역 실패가 확정됩니다. 실패하더라도 게임이 종료되지는 않습니다. 정산 결과를 수령한 뒤 Caravan을 다시 구성하여 다음 무역을 시작할 수 있습니다.

### 1차 빌드에서 실패 조건이 아닌 항목

다음 기능은 후속 빌드에서 확장할 예정이며, 1차 빌드의 일반 무역 실패 조건으로 사용하지 않습니다.

* 용병 전투 및 약탈 실패
* 로드 이벤트로 인한 운행 불가
* 계절·재난에 의한 직접적인 무역 실패

## 6. 저장 및 이어하기

게임은 무역 진행 상태를 저장합니다.

저장되는 주요 상태는 다음과 같습니다.

* 현재 Caravan 구성
* 진행 중인 무역과 무역로
* 무역 시작 시간과 종료 예정 시간
* 이동 진행 상태
* 남은 식량
* 정산 대기 상태
* 정산 수령 결과

### 진행 중 종료한 경우

* **이동 중 종료:** 재실행 후 이동 진행률과 경과 시간이 복구됩니다.
* **정산 전 종료:** 재실행 후 같은 정산 결과를 다시 확인할 수 있습니다.
* **정산 수령 후 종료:** 이미 받은 정산이 다시 지급되지 않습니다.

---

## 7. 1차 빌드 범위 안내

이번 빌드는 핵심 무역 루프와 저장·복구 구조를 검증하기 위한 Vertical Slice입니다.

다음 기능은 1차 빌드의 필수 플레이 범위에 포함되지 않으며, 2차 빌드 이후 확장할 예정입니다.

* 플레이어 및 Caravan 성장
* 계절과 재난
* 도시별 시장 갱신
* 로드 이벤트
* 용병 전투와 약탈
* 기부, 투자, 대출
* 튜토리얼과 마스코트
* 여러 Caravan의 동시 운영
* 자동 재출발 및 고급 추천 기능

---

## 8. 플레이 팁

* 식량을 너무 적게 적재하면 이동 중 무역에 실패할 수 있습니다.
* 식량을 너무 많이 적재하면 무역품을 실을 공간이 줄어듭니다.
* 적정 적재량을 넘긴 과적 상태에서는 출발할 수 있지만 이동 시간이 길어집니다.
* 무역품을 하나라도 실어야 출발이 가능합니다.
* 1차 빌드에서는 두발과 손수레만을 믿어야 합니다. 
* 정산 화면에서는 **정산 받기(Claim)**를 눌러야 수익이 실제 재화에 반영됩니다.
* 게임을 종료하기 전에 무역이 끝날 때까지 기다릴 필요는 없습니다. 이어하기를 통해 진행 상태를 복구할 수 있습니다.
